404模板–from flask import Flask, render_template, request

重定向—from flask import Flask, render_template, request, redirect

重定向就是将网页自动转向重定向，分为——301永久性重定向和302临时性重定向。

  • 301：新网址完全继承旧网址，旧网址的排名等完全清零
  • 302：对旧网址没有影响，但新网址不会有排名
